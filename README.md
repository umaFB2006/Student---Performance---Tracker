Student Performance Tracker

Project Description

Student Performance Tracker is a web-based application developed using Python and Flask. It helps teachers manage student records, assign grades, generate report cards, and analyze student performance.

Features

- Student Registration
- Add and Update Grades
- Generate Report Cards
- View Total Students
- View Pass Students
- View Fail Students
- Calculate Class Average
- Find Subject Topper
- Study Suggestions
- User Authentication (Sign Up and Sign In)

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite Database

How to Run the Project

1. Install required packages:
   
   pip install -r requirements.txt

2. Run the Flask application:
   
   python app.py

3. Open your browser and visit:
   
   http://127.0.0.1:5000/

Project Structure

- app.py - Main Flask application
- database.py - Database operations
- database.db - SQLite database
- templates/ - HTML templates
- static/ - CSS, JavaScript, and images
